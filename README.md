# 🌍 P3DX Cross-Border Data Exchange Dashboard

A modern, responsive frontend dashboard built as part of the **P3DX Frontend Task**. The application visualizes cross-border digital identity and document exchange workflows through an intuitive interface, enabling users to explore connected worlds, manage consents, monitor transactions, and view agent and locker information.

---

## ✨ Features

### 📊 Dashboard
- Executive overview with KPI cards
- Cross-world workflow visualization
- Interactive gateway cards
- Transaction timeline
- Quick statistics and status summaries

### 🌎 World Explorer
- Browse connected countries and digital ecosystems
- World-wise information cards
- Connection overview
- Responsive grid layout

### 👥 Agent & Locker Directory
- Searchable directory of agents
- Digital locker information
- Status indicators
- Organized card-based interface

### 🔗 Connection Flow
- End-to-end visualization of cross-border data movement
- Interactive workflow steps
- Clear process representation
- Gateway routing overview

### 🔐 Consent Management
- View active and pending consents
- Consent status indicators
- Permission overview
- User-friendly management interface

### 💳 Transactions
- Transaction timeline
- Transaction details panel
- Status badges
- Gateway routing information
- Compliance & security metadata
- Search and filtering interface
- Export action placeholders

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone <repository-url>
cd p3dx-frontend-task
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
